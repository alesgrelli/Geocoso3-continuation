What is Geocoso3?

It is a program that allows us to know the weather in every singe city of the world. It's written in Java and for the graphics it uses Space Builder. It uses APIs from GoogleMaps and Wunderweather in order to get all the informations needed.

Classes

The classes are made to be easy to understand, even if it's not the best implementation. To run the program needs a few minutes because of the code and the permissions to connect to the Internet.

How does it work?

Just run the program and write the name of the city you want to know the weather of and push the "Search" button, then wait for a few minutes and it's done.
